去年からアトラス作品の追加が続いています。もともと xbox game pass には龍が如くシリーズが提供されていましたが、セガと連携を更に強化しているのでしょうか。最近、スクウェアエニックスの作品が xbox game pass での提供を終了しつつあるので、新たに国産のゲームが提供されるのは嬉しいです。

評価は賛否両論らしいのですが、その後のアップデートでテンポなどいくつかの点が改善されたみたいですね。遊んでみればなんだかんだ面白いだろうと期待しています。そもそも xbox game pass で遊ぶなら金銭的な損はないですからね。

amazon での価格を見てみると元々 1 万円くらいだったのが、現在では 3000 円ほどまで値下がりしているようです。こういう場合は xbox game pass のようなサブスクに提供した方が利益になるのでしょうか。xbox game pass が続いてくれると自分としても助かるので、メーカーにとってプラスになればよいなと期待しています。

とりあえずペルソナシリーズを遊び終わっていないので早急にクリアしたいところです。
